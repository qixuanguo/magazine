# Event Magazine Flipbook

A static website that displays **one PDF magazine** in a **flipbook/page-turn** viewer. 
Specially made for one event that displays a single PDF.

## Features
- Flipbook page-turn viewing (mouse + touch swipe)
- PDF rendered to pages (PDF.js) + flip UI (StPageFlip)
- Controls: Prev/Next, Zoom In/Out, Fullscreen, Share, Download PDF
- Fonts:
  - Headings: League Spartan
  - Body: Inter
